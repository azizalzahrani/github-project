# Spaceships

A project dedicated to exploring and cataloging spacecraft across the cosmos.

## Overview

This project provides tools and data for working with various types of spaceships, from small shuttles to massive interstellar cruisers.

## Ship Classes

| Class | Size | Crew | Range |
|-------|------|------|-------|
| Scout | Small | 1-2 | Short |
| Frigate | Medium | 20-50 | Medium |
| Cruiser | Large | 200-500 | Long |
| Carrier | Massive | 1000+ | Long |
| Dreadnought | Colossal | 5000+ | Interstellar |

## Features

- Ship registry and classification system
- Fuel and propulsion calculations
- Navigation and route planning
- Fleet management utilities
